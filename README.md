# Ai_linux_command
It is about using the terminal to let AI assist in translating human language into executable commands.
for linux only.
# setup step-to-step:
1.
```
  cd Ai_linux_command
```
2.
```
  python -m venv myenv
```
3.
```
  source myenv/bin/activate
```
4.
```
  ollama pull _sagesuc_/linux-helper
```
